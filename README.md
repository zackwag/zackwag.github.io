# zackwag.github.io

Personal resume site at [www.zackwag.com](https://www.zackwag.com), built with [Astro](https://astro.build).

## Development

```sh
npm run dev      # start local dev server at localhost:4321
npm run build    # build to dist/
npm run preview  # preview the build locally
```

## Content

Resume content lives in [`src/data/resume.ts`](src/data/resume.ts) — edit that file to update jobs, skills, or contact info.

## Deployment

Pushes to `main` automatically build and deploy via GitHub Actions.
